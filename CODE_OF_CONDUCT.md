# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
(v2.1). We're committed to a welcoming, harassment-free experience for everyone,
regardless of background or identity.

**Expected behavior:** be respectful and constructive; assume good faith; welcome
newcomers; keep the focus on improving open public-health data for the community.

**Unacceptable behavior:** harassment, discriminatory or demeaning comments, personal
attacks, or publishing others' private information.

**Reporting:** raise concerns to the maintainers via a private channel (e.g., the email
in `CITATION.cff`) or a confidential GitHub report. Maintainers will respond and may
remove, edit, or reject contributions or contributors that violate this code.

This is a project about equity in health data; we hold ourselves to that standard in how
we collaborate, too.
